Marchive Archiver

Marchive is a command line utility that lets you easily archive and encrypt multiple files into a single one, in a jiffy.

Requirements:
- Windows operating system
- .NET Core 3.1 runtime needs to be installed (https://dotnet.microsoft.com/download/dotnet-core/3.1)

Usage: 
- Run Marchive.Console.exe -h in a terminal/cmd for instructions
- Example: 
  - Marchive.Console.exe archive file1.txt -p password
  - Marchive.Console.exe un-archive archive.mar -p password
